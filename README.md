Processo de Instalação do Windows XP no VirtualBox

1. Baixar e instalar o VirtualBox

Acesse o site oficial do VirtualBox e instale o programa no seu computador.

2. Criar uma nova máquina virtual

Abra o VirtualBox.
Clique em "Novo".
Nomeie a máquina como "Windows XP".
Selecione:
Tipo: Microsoft Windows
Versão: Windows XP (32-bit)

3. Configurar a memória RAM

Recomenda-se entre 512 MB e 1 GB de RAM.


4. Criar um disco rígido virtual

Escolha Criar um disco rígido virtual agora.
Formato: VDI.
Armazenamento: Dinamicamente alocado.
Tamanho: 10 GB a 20 GB.

5. Inserir a imagem ISO do Windows XP

Selecione a máquina criada.
Clique em Configurações > Armazenamento.
No ícone do CD, selecione o arquivo ISO do Windows XP.



6. Iniciar a máquina virtual

Clique em Iniciar.
O instalador do Windows XP será carregado.


7. Instalar o Windows XP

Pressione Enter para iniciar a instalação.
Aceite o contrato com F8.
Escolha o disco virtual criado.
Formate em NTFS.




8. Copiar arquivos e reiniciar

O instalador copiará os arquivos necessários.
A máquina virtual reiniciará automaticamente.


9. Configuração final

Defina idioma, data, hora e nome do computador.
Insira a chave do produto, se necessário.



10. Concluir a instalação

Após a configuração, o Windows XP iniciará pela primeira vez.
Instale os Guest Additions do VirtualBox para melhorar desempenho e resolução de tela.
